Post 2.4: Once Platonica

Proyecto en grupo donde aplicamos las figuras platonicas, en este caso, el icosaedro a la once. Para lograr esto se me ocurrío hacer un Bol de galleta con forma de medio Icosaedro abierto. Este bol es hecho con una receta de galleta de pascua, debido que es una galleta que, si se logra adherir bien al molde, se mantiene firme, visto en como se puede usar para hacer casas. Tambien, cuando se logra que el bol no tenga orificios, puede contener líquidos, helados y frutas como es visto en las fotos. La figura platónica permite crear un bol interesante sumado a lo funcional y, el objetivo, que sea comestible en la once.
